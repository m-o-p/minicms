miniCMS

miniCMS ist ein sehr kleines php-Script, das im Wesentlichen das Menu für die neue Seite des Wilheml-Schickard-Instituts generiert und sich um die "Surroundings" kümmert.

Im Allgemeinen ist es eher sinnvoll beim Redesign einer Seite auf die Uni-CI ein Content-Management-System einzusetzen!
Da dies hier explizit nicht gewünscht war aufgrund von Sicherheitsbedenken etc. entstand dieses Script.

Wenn Sie denken, dass das Script für Sie nützlich sein könnte, finden Sie hier
